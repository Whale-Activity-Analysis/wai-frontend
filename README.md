# WAI Frontend

Web-Dashboard für den Whale Activity Index (WAI).  
Implementiert mit Next.js 15, React, TailwindCSS und shadcn/ui.

## Features
- Übersicht über Whale Activity Scores
- Visualisierung historischer Daten
- API-Anbindung an das WAI Backend
- Client-/Server Components für maximale Performance

## Tech Stack
- Next.js 15
- React 18
- TailwindCSS
- shadcn/ui
- Recharts (oder ECharts)

## Entwicklung
npm install  
npm run dev

## ENV Variablen
NEXT_PUBLIC_API_URL=https://api.example.com

## Deployment
Build:
npm run build
